## 📦 FASE 1: INVESTIGACIÓN (Días 1-5)

### ☑️ Paso 1: Define tu producto
- [ ] Producto específico elegido (nada de "ropa genérica")
- [ ] Público objetivo claro (quién lo comprará)
- [ ] Precio venta estimado en tu mercado

### ☑️ Paso 2: Investiga la demanda
- [ ] Búsquedas mensuales en Google (usar Keywords Planner)
- [ ] Competencia en Amazon/marketplaces
- [ ] Tendencia: creciente, estable o decreciente

### ☑️ Paso 3: Calcula márgenes preliminares
- [ ] Precio fabricante China (estimado)
- [ ] Costos envío + aduanas (25-35% del producto)
- [ ] Margen mínimo 3x (ej: costo $10 → venta $30+)

### ☑️ Paso 4: Verifica restricciones legales
- [ ] Producto NO restringido (armas, medicinas, etc.)
- [ ] Certificaciones necesarias (CE, FDA, etc.) identificadas
- [ ] Aranceles aduaneros consultados (TARIC Europa)

### ☑️ Paso 5: Presupuesto inicial
- [ ] Capital disponible: mínimo $2,000-5,000
- [ ] Reserva para imprevistos (20% extra)

---

## 🔍 FASE 2: SOURCING PROVEEDORES (Días 6-10)

### ☑️ Paso 6: Busca proveedores en Alibaba
- [ ] Mínimo 10 proveedores contactados
- [ ] Filtros aplicados: Trade Assurance, Verified Supplier

### ☑️ Paso 7: Solicita cotizaciones
- [ ] Especificaciones técnicas claras enviadas
- [ ] Cantidades: 100, 500, 1000 unidades
- [ ] MOQ (cantidad mínima) aceptable

### ☑️ Paso 8: Compara ofertas
- [ ] Precio por unidad
- [ ] Calidad materiales
- [ ] Tiempo producción
- [ ] Términos pago (30% anticipo, 70% antes envío)

### ☑️ Paso 9: Solicita muestras
- [ ] 2-3 proveedores seleccionados
- [ ] Muestras pagadas ($50-200 + envío)
- [ ] Enviadas por DHL/FedEx (5-7 días)

### ☑️ Paso 10: Evalúa muestras
- [ ] Calidad coincide con expectativa
- [ ] Materiales correctos
- [ ] Funcionalidad 100%

---

## 💰 FASE 3: NEGOCIACIÓN Y PEDIDO (Días 11-15)

### ☑️ Paso 11: Negocia precio final
- [ ] Descuento por volumen solicitado
- [ ] Condiciones de pago acordadas
- [ ] Incoterm definido (FOB o EXW recomendado)

### ☑️ Paso 12: Revisa Proforma Invoice
- [ ] Producto correcto (nombre, modelo)
- [ ] Cantidad correcta
- [ ] Precio unitario y total
- [ ] Términos de pago claros
- [ ] Tiempo producción (lead time)

### ☑️ Paso 13: Haz el depósito (30%)
- [ ] Transferencia bancaria (más seguro)
- [ ] O Trade Assurance (protección Alibaba)
- [ ] Guarda comprobante de pago

### ☑️ Paso 14: Confirma inicio producción
- [ ] Proveedor confirma recepción de pago
- [ ] Fecha estimada de finalización confirmada
- [ ] Solicita actualizaciones semanales

### ☑️ Paso 15: Prepara documentación
- [ ] Copia pasaporte/DNI
- [ ] NIF/Tax ID
- [ ] Dirección entrega final

---

## ✅ FASE 4: CONTROL DE CALIDAD (Días 16-22)

### ☑️ Paso 16: Programa inspección QC
- [ ] Empresa inspección contratada (AsiaInspection, QIMA)
- [ ] O solicita fotos/videos detallados del 100% lote
- [ ] Fecha inspección: cuando 100% producción lista

### ☑️ Paso 17: Revisa reporte inspección
- [ ] Defectos críticos: 0 (ninguno aceptable)
- [ ] Defectos mayores: <2.5%
- [ ] Defectos menores: <4%

### ☑️ Paso 18: Aprueba o rechaza
- [ ] Si PASS → pagar balance (70%)
- [ ] Si FAIL → solicitar correcciones

### ☑️ Paso 19: Paga balance final
- [ ] 70% restante transferido
- [ ] Guarda comprobante

### ☑️ Paso 20: Solicita documentos de exportación
- [ ] Packing List
- [ ] Commercial Invoice
- [ ] Certificados (CE, test reports, etc.)
- [ ] Bill of Lading / Air Waybill

---

## 🚢 FASE 5: LOGÍSTICA Y ENVÍO (Días 23-35+)

### ☑️ Paso 21: Elige método de envío
- [ ] Marítimo (FCL o LCL): 25-40 días, más económico
- [ ] Aéreo: 5-10 días, más caro
- [ ] Express (DHL/FedEx): 3-5 días, muy caro

### ☑️ Paso 22: Contrata transitario/freight forwarder
- [ ] Solicita cotización a 3 empresas
- [ ] Compara: precio, servicios, opiniones
- [ ] Contrata y da instrucciones de recogida

### ☑️ Paso 23: Tracking del envío
- [ ] Número de tracking recibido
- [ ] Seguimiento en línea activo
- [ ] Fecha estimada llegada puerto/aeropuerto

### ☑️ Paso 24: Prepara despacho aduanas
- [ ] Transitario gestiona aduanas (recomendado)
- [ ] O agente aduanas propio contratado
- [ ] Documentos enviados: factura, packing list, BL

### ☑️ Paso 25: Paga aranceles e impuestos
- [ ] IVA (21% Europa, varía por país)
- [ ] Aranceles (0-20% según producto)
- [ ] Tasas aduanas (~$100-300)

---

## 📦 FASE 6: RECEPCIÓN Y VENTA (Días 36+)

### ☑️ Paso 26: Recibe mercancía
- [ ] Inspección visual cajas (sin daños externos)
- [ ] Cuenta bultos (coincide con packing list)
- [ ] Firma recepción al transportista

### ☑️ Paso 27: Inspección final
- [ ] Abre cajas, verifica contenido
- [ ] Revisa que productos coinciden con pedido
- [ ] Separa unidades defectuosas (si hay)

### ☑️ Paso 28: Prepara para venta
- [ ] Fotografía productos
- [ ] Crea listings (Amazon, tu web, etc.)
- [ ] Define precio venta final

### ☑️ Paso 29: Lanza al mercado
- [ ] Publica en canales de venta
- [ ] Marketing inicial (ads, redes, etc.)
- [ ] Primeras ventas

### ☑️ Paso 30: Analiza resultados
- [ ] Margen real vs. estimado
- [ ] Tiempo total del proceso
- [ ] Lecciones aprendidas
- [ ] Decisión: repetir pedido o ajustar

## 🚨 ERRORES COMUNES A EVITAR

### ❌ NO HAGAS ESTO:
1. **Pagar 100% por adelantado** → Paga siempre 30-70 (nunca todo antes de inspeccionar)
2. **Saltar inspección QC** → Ahorras $300, pierdes $3,000 en defectos
3. **No verificar certificaciones** → Aduanas retiene tu mercancía
4. **Elegir proveedor más barato** → Barato sale caro (calidad mala)
5. **No tener seguro** → Contenedor se pierde, pierdes todo
6. **Olvidar IVA/aranceles en cálculo** → Margen real es 0
7. **No negociar términos de pago** → Quedas vulnerable
8. **Confiar solo en fotos del proveedor** → Siempre inspecciona

### ✅ HAZ ESTO:
1. **Empieza con pedido pequeño** (100-500 uds) para probar proveedor
2. **Pide muestras SIEMPRE** antes de pedido grande
3. **Usa Trade Assurance** en Alibaba (protección comprador)
4. **Documenta TODO** (emails, contratos, fotos)
5. **Ten proveedor alternativo** (backup si uno falla)
6. **Calcula costos reales** ANTES de pedir
7. **Lee contratos/facturas** antes de firmar/pagar
8. **Mantén comunicación constante** con proveedor

## 🎯 PRÓXIMOS PASOS

### Después de tu primera importación exitosa:

1. **Optimiza el proceso**
   - Negocia mejores precios por volumen
   - Reduce tiempos de producción
   - Mejora embalaje

2. **Diversifica proveedores**
   - Ten 2-3 proveedores para mismo producto
   - Evita dependencia de uno solo

3. **Amplía tu catálogo**
   - Añade productos complementarios
   - Usa mismo proveedor si es bueno

4. **Sistematiza**
   - Crea SOPs (procedimientos estándar)
   - Automatiza seguimiento
   - Delega tareas repetitivas
