## RESTAURAR

1. Extraer backup:
   tar -xzf importar-china-backup-YYYYMMDD-HHMMSS.tar.gz

2. Instalar dependencias:
   cd importar-china-web
   npm install

3. Configurar .env.local:
   - STRIPE_SECRET_KEY=sk_live_...
   - NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_...
   - RESEND_API_KEY=re_...
   - NEXT_PUBLIC_BASE_URL=https://importardechina.com

4. Build:
   npm run build

5. Deploy:
   export VERCEL_TOKEN=...
   npx vercel --prod --yes
